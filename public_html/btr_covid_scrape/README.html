What

Scrapes todays (or latest) Manitoba covid19 stats for daily cases and transmission rates and adds them to a mysql database. Then gets all values from the database and prints them.

How :

I used a webscraping service, scraperapi to scrape the manitoba provincial goverment daily covid stats, found at https://www.gov.mb.ca/covid19/updates/index.html.

This returned the full html webpage. Everything could be discarded except the current situation content. Then the current situation text was parsed for the date, the daily # of covid cases, winnnipeg transmission rate and provincial transmission rate.

These values were then inserted into a mysql database, and the entire record of the database is then printed out to the screen.

Pivot
